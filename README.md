# Railway Reservation System

A console-based railway ticket reservation system built with Python.

## Features

### Passenger Features
- User registration with PNR generation
- Login system
- Seat availability checking
- Ticket booking
- Booking status viewing

### Cashier Features
- View all passenger details
- Approve ticket bookings
- Cancel tickets
- Manage seat allocation
